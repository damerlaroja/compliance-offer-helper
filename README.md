# Compliance-Friendly Offer Helper
Built for the Amazon Nova AI Hackathon 2026.

Helps fintech and commerce teams generate compliant customer-facing 
messages using a two-agent AI pipeline powered by Amazon Nova on AWS Bedrock.

## How it works
1. Drafter Agent: Creates customer-facing message with key disclosures
2. Reviewer Agent: Checks compliance and flags missing terms

## Built with
- Amazon Nova Lite via AWS Bedrock Converse API
- Python + Streamlit
- Prompt injection protection + input hardening
